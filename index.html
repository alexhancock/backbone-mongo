Backbone.MongoCollection

Download Github

Query your Backbone.Collections with the MongoDB API, client or server side.

This project is glue between...

  • Minimongo - by the Meteor team
  • Backbone.Collections

It does just what you think it does...

You can run mongo queries on your Backbone.Collections, even in your clientside code. Queries run synchronously on a completely in memory implementation of mongo.

Queries

# Use any mongo query you like
collection.find({ score: { $gt: 5 }});
collection.find({ name: "Alex" });

# Inserts and updates work as well
collection.insert({ name: "Claire" });
collection.update({ name: "Alex" }, { $inc: { score: 10 } });
